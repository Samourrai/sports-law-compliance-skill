## TIER 1 — Global Bodies

### CAS/TAS — All Sports
- **Abbreviation**: CAS / TAS
- **Type**: Global arbitration
- **Website**: cas-crt.org
- **Key regulations**: CAS Code of Sports-related Arbitration (2024)
- **Dispute resolution path**: Final instance for all international sports disputes
- **Reference file**: `references/cas-tas.md`
- **Version date**: January 2024
- **Official regulations URL**: https://www.cas-crt.org/en/arbitration-rules/

### WADA — Anti-Doping (All Sports)
- **Abbreviation**: WADA
- **Type**: Global
- **Website**: wada-ama.org
- **Key regulations**: World Anti-Doping Code 2021, Prohibited List (updated annually Jan 1)
- **Dispute resolution path**: IF internal → CAS
- **Reference file**: `references/olympic-wada.md`
- **Version date**: January 2021 (Code); January 2025 (Prohibited List)
- **Official regulations URL**: https://www.wada-ama.org/en/prohibited-list

### IOC — Olympics
- **Abbreviation**: IOC
- **Type**: Global
- **Website**: olympics.com / ioc.org
- **Key regulations**: Olympic Charter, Host City Contract
- **Dispute resolution path**: IF internal → CAS Ad Hoc (during Games) / CAS Appeals (outside Games)
- **Reference file**: `references/olympic-wada.md`
- **Version date**: 2021
- **Official regulations URL**: https://olympics.com/ioc/olympic-charter
